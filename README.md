# 🛒 Online Shop Project (C Language)
### 👥 Team CatalystS

## 📌 Project Overview
The Online Shop Project is a console-based application built using the C programming language. It simulates a simple e-commerce system where users can browse products, add items to a cart, and place orders.

This project demonstrates core programming concepts such as structures, functions, loops, arrays, and file handling.
